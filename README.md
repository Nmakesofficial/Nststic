# Nstatic — AI-Powered Static Website Generator
**Created by Nmakes**

Nstatic is a modern, lightweight, AI-assisted static website generator designed to help developers, designers, and creators build fast, scalable websites with minimal effort.  
It combines automated content generation with a clean build pipeline, giving you full control over structure, styling, and deployment — without the complexity of large frameworks.

---

## ✨ Features

### **AI-Generated Static Pages**
Automatically generate full HTML/CSS pages using built-in AI templates.

### **Ultra-Fast Build System**
Simple, dependency-light compiler optimized for static output.

### **Customizable Templates**
Modify layouts, components, and styles to create unique designs.

### **Zero Backend Needed**
Everything compiles to pure static files — HTML, CSS, JS.

### **SEO-Friendly Output**
Clean markup, optimized metadata, and standardized structure.

### **Lightweight & Modular**
No heavy frameworks — easy to integrate into any workflow.
